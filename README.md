## Goods_Expiry_Date_Tracker

## Features:
- Scan QR Code
- Get Scanned data, save it in database and display it in a recyclerView
- when item's date expires, show notification


## Tools :       
* Kotlin
* MVVM
* Room db
* viewModel , LiveData
* Coroutines
* Navigation component 
* Hilt
* ViewBinding
* RecyclerView
